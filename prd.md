# Project Brief: LinguistAI

## 1. Project Overview

**LinguistAI** is a sophisticated, AI-native language learning and translation platform. Unlike traditional translators, LinguistAI focuses on deep contextual understanding and retention, helping users not just translate text but master the nuances of their target language through integrated study tools.

## 2. Core Value Proposition

- **Precision Translation:** AI-powered translation with contextual toggles (Formal/Informal).
- **Educational Depth:** Automatic word-by-word breakdown with grammatical tagging, phonetics, and usage examples.
- **Mastery Loop:** A seamless flow from translation to Glossary (storage) to Flashcards (active recall).

## 3. User Experience & Design Language

- **Brand Identity:** Professional, clean, and trustworthy.
- **Design System:** _Synthetica Design Language_ (Light Mode).
- **Key UI Patterns:**
  - Persistent sidebar for global navigation.
  - Component-based card layouts for vocabulary.
  - High-contrast typography (Inter) for readability.
  - Soft surface colors (#f8f9ff) with primary blue (#2563eb) accents.

## 4. Feature Set (Current State)

- **Translator Dashboard:** Input area with language selection, formal/informal toggles, and multi-word breakdown cards.
- **Word Breakdown:** Detailed cards showing word class, pronunciation, definition, and contextual usage examples.
- **Glossary / Saved Words:** A searchable, filterable repository of favorited vocabulary with learning status tracking.
- **Flashcards Study Interface:** A focused, minimalist study mode utilizing spaced repetition concepts (Mastered, Getting There, Still Learning).
- **Collapsible Navigation:** A responsive sidebar designed to maximize workspace efficiency.

## 5. Technical Requirements

- **Responsive Design:** Optimized for Desktop (current) with mobile-ready navigation patterns.
- **Accessibility:** High-contrast text and clear iconography for audio pronunciation and actions.
- **Interactivity:** DOM-based state management for card flipping, filtering, and navigation.

## 6. Future Roadmap

- Mobile-friendly responsive views.
- Translation history management.
- End-of-session study summaries and accuracy metrics.
- User profile and achievement tracking.
