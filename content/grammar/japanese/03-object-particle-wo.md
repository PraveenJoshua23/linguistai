id: object-particle-wo
title: Object Particle を
level: Beginner
summary: 'を marks the direct object of a verb, pronounced "o".'
examples:
  - text: 手紙を書きます。
    romanization: tegami o kakimasu.
    translation: I write a letter.
  - text: 音楽を聞きます。
    romanization: ongaku o kikimasu.
    translation: I listen to music.
  - text: お茶を飲みます。
    romanization: ocha o nomimasu.
    translation: I drink tea.

を marks the direct object, the thing an action is done to. It is written を but pronounced "o". The pattern is "[object] を [verb]".
